Emulationstation Gamelist Manager
=================================

Download on Recalbox
--------------------

```bash
wget https://github.com/neolao/emulationstation-gamelist-manager/releases/download/latest/egm-linux-armv7 -O egm
chmod +x egm
```

Commands
--------

### Check gamelist
```bash
./egm check /path/to/gamelist.xml
```
```
Number of games: 42
Not found: 3
```
